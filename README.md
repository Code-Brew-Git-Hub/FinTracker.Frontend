# 🎨 FinTracker.Frontend

Frontend часть сервиса **FinTracker** — интерфейс для отслеживания и анализа финансовых трат.

---

## 🚀 О проекте

Данный репозиторий содержит клиентскую часть приложения:

* Главная страница
* UI компоненты
* Визуализация данных (в будущем)

---

## 🛠 Используемые технологии

* HTML5
* CSS3
* JavaScript (Vanilla)

## 📸 Текущий статус

🔹 Реализована:

* Главная страница (Hero-блок)

🔹 В разработке:

* Графики и аналитика
